Introdução

O HTML, sigla para Linguagem de Marcação de Hipertexto, é a base da construção de páginas da web.

Com HTML, você descreve a estrutura do conteúdo para que o navegador saiba exibir corretamente cada parte. Por exemplo: o título, o parágrafo, o link, a imagem… Tudo isso é marcado com as chamadas tags.

Curiosidade: O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação. Ele não executa lógica, apenas estrutura as informações no navegador, através de tags que definem o conteúdo.

Aula 1: HTML na Prática

Você não precisa instalar nada. Vamos começar com o que já vem no seu computador.

Abra o Bloco de Notas e crie sua primeira página web em 4 passos:

Passo 1: Pressione Windows + R, digite notepad e aperte Enter.

Passo 2: Digite o seguinte código:

<!DOCTYPE html>
<html>
  <head>
    <title>Minha Primeira Página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Minha primeira página HTML criada no bloco de notas.</p>
  </body>
</html>

Passo 3: Salve como index.html. Na opção \"Tipo\", selecione Todos os Arquivos.

Passo 4: Dê dois cliques no arquivo salvo e abra no navegador.

Importante: A extensão .html é essencial para o navegador reconhecer o arquivo como página da web.

Importante: Digite o código não cópie e cole. Assim você aprende melhor. Se errar, não tem problema, é assim que se aprende!

Aula 2: Trabalhando com Formulários

Formulários servem para coletar dados do usuário. No HTML usamos a tag <form> e dentro dela colocamos campos como:

  • <input> – Campo de texto, e-mail, senha, etc.
  • <button> – Botão para envio
  • <checkbox> – Caixa de seleção
  • <radio> – Seleção única
  • <select> – Lista de opções
  • <textarea> – Campo de texto longo

Os métodos GET e POST definem como os dados são enviados para o servidor. O GET mostra as informações na URL (tipo uma carta aberta). O POST envia os dados de forma mais segura (tipo carta fechada).

Aula 3: Estruturando o HTML + Formatação

Nessa etapa aprendemos a organizar e estilizar o conteúdo usando:

  • <strong> – Negrito semântico (importância)
  • <b> – Negrito visual
  • <i> – Itálico
  • <u> – Sublinhado
  • <div> – Container de bloco
  • <span> – Container em linha
  • <fieldset> – Agrupa campos de formulário
  • <embed> e <iframe> – Incorporam conteúdos externos

Também vimos como usar cores: nomes, hexadecimal e RGB.

Aula 4: Trabalhando com Mídias

O HTML permite adicionar mídias como imagens, áudio e vídeo com as tags:

  • <img> – Imagem
  • <audio> – Áudio
  • <video> – Vídeo
  • <track> – Legenda de vídeo
  • <iframe> – Conteúdo de outro site (ex: YouTube)

Aula 5: Tabelas

As tabelas são úteis para exibir dados em linhas e colunas:

  • <table> – Inicia a tabela
  • <tr> – Linha da tabela
  • <td> – Célula comum
  • <th> – Cabeçalho
  • <thead>, <tbody>, <tfoot> – Cabeçalho, corpo e rodapé

Lembre-se: tabela não é para layout, é para dados.

Aula 6: HTML Semântico

O HTML semântico melhora a organização e a acessibilidade das páginas.

Principais tags:

  • <main> – Conteúdo principal
  • <header> – Cabeçalho
  • <footer> – Rodapé
  • <nav> – Navegação
  • <section> – Seção
  • <article> – Conteúdo independente
  • <aside> – Conteúdo complementar
  • <figure> e <figcaption> – Imagem com legenda
  • <blockquote> e <q> – Citações
  • <picture> – Imagens adaptativas
